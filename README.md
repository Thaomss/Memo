# Memo Drago

Petite application web statique pour le Domaine de la Dragonniere :

- recap des horaires utiles pour le service ;
- plan interactif avec recherche par numero de logement ou d'emplacement ;
- recap rapide des supplements 2026.

## Lancer en local

Prerequis : Node.js 20 ou plus recent.

```bash
npm install
npm run dev
```

## Verifier

```bash
npm run build
```

## Publier sur GitHub Pages

Le projet est compatible avec GitHub Pages via Vite.

```bash
npm run deploy
```

Dans les reglages du depot GitHub, activez ensuite Pages sur la branche `gh-pages`.

## Notes

Les donnees viennent des affichages fournis en photo. Les horaires marques "sur place" doivent rester verifies avec l'affichage officiel du domaine.
